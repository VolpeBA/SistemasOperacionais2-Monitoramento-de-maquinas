- Projeto desenvolvido para aprovação na matéria de Sistemas Operacionais II no curso de Análise e Desenvolvimento de Sistema na Fatec Americana.

# Sobre o projeto

- Frameworks utilizados Telegraf, Influxdb e Grafana.
